v1.1.4
Patch Schedule Mode 2
Added day-of-week patch scheduling — select specific days (e.g. Monday & Thursday) instead of every X days. Badge appears on scheduled days, escalates to terracotta if not confirmed by the next day. Both modes supported and switchable in settings. Push notifications updated to fire on each selected day.
v1.1.3
Housekeeping
Removed haptic feedback from the roadmap — not supported on iOS. Backlog cleanup.
v1.1.2
Patch Tracking Cleanup & UX Polish
Unified patch change tracking to a single consistent path — "I changed my patch today" button on landing and notes screen are now the only entry points, eliminating potential data inconsistency. Landing shortcut restyled as a soft rose card with icon to match the app aesthetic. Auto-advance delay tuned to 800ms.
v1.1.1
Side Effects, UX Polish & Auth
Split event chips into two groups — Events and Side Effects. Added Nausea, Fatigue, Patch irritation, and Bloating as side effect chips. Side effects stored separately from events in the database. Auto-advance delay increased to 2 seconds. Feedback confirmation shows for 2 seconds. Password now remembered for one year so she only needs to log in once.
v1.1.0
Settings, Notifications & Patch Tracker
Full settings screen with auto-save. Patch change tracker with day-of badge and terracotta urgency escalation. Push notifications for patch reminders, daily check-in, and weekly summary — all configurable with time pickers. Landing screen shortcut to log patch changes without a full check-in. One-time onboarding nudge for PWA home screen install. Note: notifications require the app to be installed via Safari to home screen and may occasionally be delayed on iOS.
v1.0.7
UX & Stability
Auto-advance after rating selection with a 600ms delay — cancellable by tapping Back. Added Pass button to skip a question without recording a score. Scale hints updated to "1 - rough day" and "feeling great - 5". Fixed date handling across streak dots and query ranges to use device local time correctly. Hot flash question reworded and moved to last position.
v1.0.6
Questions & Events Update
Added eczema tracking — skin clarity rated 1–5 daily. Replaced the Poor sleep event chip with Intimacy. Step count noted for future structured tracking.
v1.0.5
Annotation & Events
Added a Patch Applied button and expandable Mark an Event section to the notes screen. Includes 8 preset event chips and a custom option. All selections are toggleable and restored when editing today's entry.
v1.0.4
PWA & Home Screen Icon
Added iOS home screen support. The app can be saved to iPhone via Safari with a custom rose icon, full-screen mode, and proper app title.
v1.0.3
1–5 Rating Scale
Simplified daily ratings from 1–10 to 1–5 for faster, friendlier entry on mobile.
v1.0.2
Table Naming & Bug Fixes
Renamed Supabase tables to the ht_ prefix for multi-app organisation. Fixed question and log loading that was preventing check-in from starting.
v1.0.1
Security Hardening
Tightened Supabase row-level security so historical entries cannot be modified — only today's entry is editable via the anon key.
v1.0.0
Initial Release
Daily wellness check-in with 10 questions, 1–5 tap ratings, progress bar, 14-day streak tracker, optional notes, and trend charts with 7/30/90 day windows and per-metric drilldown.
